Why Flutter?

  1. Faster Development with hot reload by default
  2. Intiutive and extremely flexible UI
  3. Performance / native and so fast
  4. By Default AOT thanks to Dart language
  5. Amazing documentation and community

What will we get from this workshop?

  1. Learn how to initalize and create a project
  2. Get the basics with some examples
  3. Real life examples from the apps that I have on both App Stores
  4. Code bites (yummy)
  5. I will try to reply every single question.

What is the goal of this workshop?

  1. Seeing how easy to setup a mobile app for multiple platforms
  2. Being prepared on any mobile need of our clients
  3. Of course to share knowledge
  4. And get some fun, if you are ready.
